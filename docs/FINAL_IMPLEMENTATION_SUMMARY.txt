DATABASE SETUP REQUIRED:
================================================================================

IMPORTANT: You must run the SQL commands from Database/database_extensions.txt
in your MySQL database before the website will work properly.

Steps:
1. Open phpMyAdmin or MySQL command line
2. Select your database (elegance-salon)
3. Run all CREATE TABLE statements from database_extensions.txt
4. Run the ALTER TABLE statement for appointment_notifications
5. Create all indexes as specified
